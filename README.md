*CUIRPG
UNIX系のCUIでゲームを作りたいという私みたいなおろかな考えをお餅の方へこれを渡します。
sankou.txtに参考にさせていただいたサイトのURLが書いてあります。
これは使いようによってはRPGにもSTGにも横スクロールアクションにもできます。
改造の仕方は人それぞれです。
ゲームに必要最低限な

・画面描画(がめんびょうが)
・キー入力
・タイマー割り込み

この三つの関数が一緒になっています。

このソースコードの全部や一部をそのままや改変してソフトを作る場合、オープンソースだろうがクローズだろうが明記の必要はないです。
このソースコードをコンパイルせず改変(新しい機能付けたり)してソースコードを配布する際は、強制ではありませんがテキトーに書いてもらえるとうれしいです。書かなくてもいいです。

他にも様々なサイト様を参考にさせていただいているので、もしここのソースコードは自分が作ったものだ!と言う方がいらっしゃればご連絡ください。明記いたします。
